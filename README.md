# Eventizo

Blockchain-based Event Ticket Management System.

## Overview
Eventizo is a web-based event ticket management platform that integrates Blockchain Smart Contracts to ensure transparency, security, and integrity in ticket transactions.

## Features
- Event management
- Online ticket purchasing
- Ticket transfer between users
- Real-time chat between users and admin
- Payment integration using PayOS
- Smart Contract verification for ticket transactions
- Admin dashboard for managing events, users, and tickets

## Technologies

Backend
- ASP.NET Core
- RESTful API
- SignalR
- JWT Authentication

Frontend
- ASP.NET MVC

Database
- SQL Server

Blockchain
- Solidity
- Ganache
- Remix

Payment
- PayOS
